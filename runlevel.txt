production servers runs of runlevel 3 or 5. 
3 is cli mode
5 is gui mode 
there are total 7 modes (0-6) in runlevels
0 is for shutdown 
6 is for reboot
3 is for cli mode
5 is for gui mode
2 is for cli without network
4 is blank
1 is for single usermode
